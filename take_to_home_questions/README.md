

Test setup using Cucumber.js and JavaScript.

##  Requirements

- Node.js (v14 or higher)
- npm (comes with Node.js)


##  Clone the Project

```bash
git clone https://github.com/hazaelmartinez/cucumberTest.git
cd cucumberTest

##  Setup

Install the dependencies:

```bash
npm install


Run Tests
```bash
cd .\take_to_home_questions\
npx cucumber-js
